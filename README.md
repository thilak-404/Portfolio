# Professional AI & ML Portfolio - Thilak Divyadharshan

This is a premium, mobile-responsive portfolio website designed for **Thilak Divyadharshan**, a B.Tech CSE (AI & ML) student.

## Features
- **Neural Midnight Theme**: Sleek dark design with glassmorphism.
- **Micro-animations**: Smooth reveal-on-scroll and typing effects.
- **Mobile Responsive**: Fully optimized for phones, tablets, and desktops.
- **Modern Tech**: Built with HTML5, CSS3 (Vanilla), and ES6+ JavaScript.

## How to View the Site

### Option 1: Direct Open
Simply double-click the `index.html` file in your file manager to open it in your default web browser.

### Option 2: Using a Local Server (Recommended)
Running through a local server ensures all assets and animations load perfectly.

1. Open a terminal in this directory.
2. Run the following command:
   ```bash
   python3 -m http.server 8000
   ```
3. Open your browser and visit:
   [http://localhost:8000](http://localhost:8000)

## Project Structure
- `index.html`: Main structure and content.
- `style.css`: Premium styling and animations.
- `script.js`: Interactivity and logic.
- `assets/img/`: Optimized images and banners.
